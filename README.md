# Visual_Question_Answering_-VQA-
Visual Question Answering (VQA) là một lĩnh vực trong trí tuệ nhân tạo (AI), nơi hệ thống máy tính được yêu cầu trả lời các câu hỏi liên quan đến một hình ảnh. VQA kết hợp giữa computer vision và natural language processing (ví dụ: "Có bao nhiêu người trong bức tranh?" ) và câu trả lời phù hợp (chẳng hạn như "3") .
